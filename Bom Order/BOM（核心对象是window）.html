BOM（核心对象是window）

window.open(URL,name,specs)

 作用：打开新窗口；参数：String

 返回值：新打开窗口的window对象

 注意：写在全局会被浏览器拦截，不传参数默认新窗口打开

 URL地址（string）

  'http://www.sina.com',不写http默认为本地地址

 name:打开方式，或者是框架name

  <iframe name="iframe1">

 _blank:新窗口打开；_self：当前窗口打开；_top：最外层框架；_parent：父级框架

 specs:规格，只有在新窗口打开有效，'width=100px,height=100px',属性=值，多个属性间用逗号分隔

window.close();

 作用：关闭窗口

 可以用变量先记录打开测新窗口，然后再用close()关闭

 firefox下只能关闭open打开的窗口

navigator

 浏览器信息对象

 navigator.userAgent

  string,浏览器信息，什么浏览器Chrome，Firefox，版本号

location

 地址栏对象,下面的属性可以获取也可以设置

 location.href

  绝对地址，修改值后会刷新页面

 location.search

  查询信息，修改值后会刷新页面

  search设置不能放到全局，不然会无限刷新页面

  ？s=abc &符号连接多个

 location.hash

  锚信息，不会刷新页面

  window.onhashchange 当hash改变触发的事件

  #page=1 &符号连接多个

window.top / window.parent

 最外层window对象 / 父级window对象

window.innerWidth / window.innerHeight

 获取可视宽高 - 标准方法

 高版本支持，低版本不支持

 PC端可用下面这种方法，兼容性比较好

 document.documentElement.clientWidth / Height

window.onresize

 窗口尺寸改变时候触发事件

window.onscroll

 滚动条滚动时触发的事件

window.pageXoffset / window.pageYoffset

 获取滚动条滚动距离 - 标准方法

 chrome方法

  document.body.scrollTop

 其他浏览器方法

  document.documentElement.scrollTop

  chrome也可以用这种方法，但是获取到的是0

window.scrollTo(x,y)

 设置滚动条滚动距离

 参数：x轴距离，y轴距离

 （0，1000）参数不带单位

scrollHeight / scrollWidth

 例：console.log(box.scrollHeight)

 如果自己的内容 或子元素没有超出本身，为clientHeight的值

 如果自己的内容或是子元素超出了本身，为自身clientHeight 值 + 超出部分高度
