DOM

document.createElement('标签名')

 创建元素，create是document下的方法，所以前面必须这么写

 只是创建了元素，但是页面上还没有这个元素，

父级节点.appendChild(要插入的节点)

 作用：向父级节点！！最后！！插入要插入的节点

 如果插入的是页面中已有的节点，相当于！！剪切！！效果

父级节点.insertBefore(要插入的节点，插在哪个节点前)

 作用:在父级节点下，把要插入的节点，插入到被插入的节点前

 如果插入的是页面已有节点，相当于！！剪切！！

 被插入的节点一定要是父级节点下的节点，否则会！！报错

父级节点.removeChild(要删除的节点)

 删除父级节点下要删除的节点

 要删除的节点必须在父级节点下 ！！！否则会报错

父级节点.replaceChild(要替换的节点，被替换的节点)

 在父级节点下，用要替换的节点，替换被替换的节点

 被替换的节点一定要在父级节点下 ！！！否则会报错

 如果要替换的节点是页面中已有的节点，相当于！！剪切！！

node.cloneNode(参数)

 克隆节点，前面是要被克隆的节点

 不传参数默认是false

 true:深度克隆，子节点也克隆，包括行间属性

 false:只能克隆自身包括行间属性，下面的子节点不克隆

 可以克隆 写在行间的 属性和自定义属性

 不能克隆：JS添加的自定义属性,和事件

页面可视区宽/高

 document.documentElement.clientWidth

 document.documentElement.clientHeight

E.clientWidth / E.clientHeight

 自身宽/高+padding，不带单位

 内嵌元素 是 0

E.offsetWidth / E.offsetHeight

 自身宽/高+padding+border，不带单位

E.offsetLeft / E.offsetTop

 获取 E 左 / 上 边框外，到E的定位父级的 左 / 上 边框内的距离，不带单位

E.offsetParent

 获取E的定位父级元素,属性值不为static

 没有定位父级，最后找到body

 body的定位父级是null

this.herf

 当前链接的href属性 == this.getAttribute('href')

node.nodeType

 查看节点类型，返回值Number

 常用的节点类型 1 - 元素节点，2 - 属性节点，3 - 文本节点，8 - 注释节点，9 - document节点

node.nodeName

 查看节点名称，返回值（string）

E.parentNode

 获取该元素节点的父级元素节点（只能获取，不能设置）

E.previousElementSibling

 获取上一个兄弟元素节点

 被返回的节点以 Node 对象的形式返回

 如果没有，则返回值是 null

 同样也有 prevSibling

E.nextElementSibling

 获取下一个兄弟元素节点

 同样也有 nextSibling

E.childNodes

 获取到所有第一级的子节点（包括空格 和换行符 算作文本节点）

 E.children - 获取所有第一级元素节点,是个集合（非标准方法，厂商支持）

E.firstElementChild

 获取E元素节点下第一个元素节点

 E.firstChild - 可以获取E元素下第一个任意类型的节点

E.lastElementChild

 获取E元素节点下最后一个元素子节点

 E.lastChild - 可以获取E元素节点下第一任意类型的节点

E.removeAttribute('属性名')

 删除E元素节点的行间属性和属性值

E.setAttribute('属性名'，('属性值'))

 行间固有属性和行间自定义属性都能设置

E.getAttribute('属性名')

 获取行间固有属性，和自定义属性

E.getBoundingClientRect（）

 对象，top bottom left right width height ，不带单位

 bottom - 元素节点的下外边框到！！！可视区上边的距离

 right - 元素右外边框到！！！可视区左边的距离

E.attributes

 获取到的是元素属性集合
